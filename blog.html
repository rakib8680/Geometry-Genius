!! Important Questions !!

  1. 1. What is DOM and what is the purpose of DOM?

    2. Ans: The full form of DOM is : document object model. The purpose of dom is to make a website dynamic and interactive,
    to use this it has to be connected with javascript. With DOM, we can change the font, size,
    color, background etc dynamically .

  2. 2. How will you select HTML elements using DOM? Name the DOM methods.

    3. Ans: We can select dom in various ways. There are five common ways to select HTML elements using dom :

    • 1. getElementById()
    • 2. getElementByClassName()
    • 3. getElementByTagName()
    • 4. querySelector()
    • 5. querySelectorAll()

  3. 3. What is event bubble?

    4. Ans: Event Bubble is a very important thing in Dom. It happens when elements clicked or when an elements receive an
    events and bubble propagate to its parent. And it's propagate until it get to the root element.

  4. 4. What is a callback function and why will you use it?

    5. Ans: A callback function is any reference to executable code that is passed as an argument to another piece of code.
    When we need to use one function to execute only after another function has completed execution
    then we use call back function. We can use this in Javascript
